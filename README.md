# [day-night-cycle](https://github.com/UniBreakfast/day-night-cycle)

## An animated sky scene with the Sun and Moon moving around it

### You can control the speed and current phase by moving the mouse cursor up/down. Horizontal position does not matter. Top part of screen notices the mouse after delay (on purpose).
